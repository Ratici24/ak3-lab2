# Лабораторна робота №2 — Основні інструкції 32-бітного ARM

**Дисципліна:** Архітектура Комп'ютерів. Частина 3
**Виконав:** Миркін Радислав Ігорович, ІО-31, залікова №3114
**Кафедра обчислювальної техніки, ФІОТ, КПІ ім. Ігоря Сікорського**

## Варіант 4 (3114 mod 5)

```
           ┌─  (a | b) · c,   якщо (a · b) ≥ 10
f(a,b,c) = │
           └─  (a · b) · c,   якщо (a · b) <  10
```

Тестові значення: a=4, b=3, c=5 → f = (4|3)·5 = 7·5 = 35 (0x23)

## Структура

| Файл | Призначення |
|------|------|
| `start.S` | Vector table, Reset Handler, виклик lab1 |
| `lab1.S` | Реалізація функції за варіантом 4 |
| `lscript.ld` | Скрипт лінкера (Flash 1MB / RAM 128K) |
| `Makefile` | Автоматизація збирання |

## Збирання

```bash
make           # firmware.elf + firmware.bin
make qemu      # запуск у qemu з gdb-сервером на tcp::1234
make clean     # прибрати артефакти
```

## Відлагодження

```bash
gdb-multiarch firmware.elf
(gdb) target extended-remote 172.18.224.1:1234
(gdb) break _loop
(gdb) continue
(gdb) info registers   # r0 = 0x23 = 35
```
